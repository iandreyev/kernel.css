Built for any kind of job

Whether you're making a website, web app or a dashboard app, this framework got you covered!

Far from reaching it's potential

If you like this project and would like to help out, you may do so on github. This project is ever improving.

Simple to use

The Kernel Framework is designed for everyone from the beginner to the novience. It makes everybody's job easier.

Lightweight

We aim to make this a super lightweight, fully fledged framework that wont slow down your website's loading times.

A solid foundation for your project

The Kernel Framework does not aim to do all the work for you by any means, it is meant to be a generic foundation that can be used for any type of project.

Compatible with other frameworks

Because kernel.css uses prefixes for all it's css classes (ion-), it can be used with most other CSS framework. Mix and match

The ion-base class contains the styles for typography, headings and paragraphs.

Welcome

    <h3>Welcome</h3>
</div>

You may use the "ion-vertical-center" class to vertically center full height hero's.

Example

<div class="ion-hero" style="height: 100vh;">
    <div class="ion-container ion-vertical-center">
        <h3>Welcome</h3>
    </div>
</div>

Inputs

<!-- Default Button -->
<button class="ion-button btn-default">Button</button>

<!-- Primary Button -->
<button class="ion-button btn-primary">Button</button>

<!-- Safe Button -->
<button class="ion-button btn-safe">Button</button>

<!-- Warning Button -->
<button class="ion-button btn-warning">Button</button>

<!-- Risk Button -->
<button class="ion-button btn-risk">Button</button>

<!-- Minimal Button -->
<button class="ion-button btn-minimal">Button</button>

<!-- Large button -->
Just append the "ion-btn-lg" class to any of the button types.

You may also use the "ion-button" class with anchor tags and button inputs,
and you can extend the class if you want to customize your own buttons.

Notices

Use notices to inform your users of something important. There are multiple types of notices; ion-info, ion-warning and ion-alert.

<div class="ion-notice ion-info">This is a notice.</div>
<div class="ion-notice ion-warning">This is a warning.</div>
<div class="ion-notice ion-info">This is an alert!</div>
Notice: This is a notice.close
Warning: This is a warning.close
Alert: This is an alert!close

Colors

<div class="ion-bg-{color}"></div>

kernel.css includes predefined color classes. These are great for prototyping your ideas quickly, but you may not want to pollute your dom with a bunch of color classes.

Utils

kernel.css comes bundled with a few useful utility classes.

<!-- Anchor -->
<a class="ion-anchor" href="#">

<!-- Horizontal Rule (hr) -->
<hr class="ion-hr">

<!-- Section -->
Gives a set margin to an element. Used to seperate content.
<div class="ion-section"></div>

<!-- No-shadow -->
Removes all shadows from an element.
<div class="ion-noshadow"></div>

<!-- Vertical-center -->
Vertically center an element.
<div class="ion-vertical-center"></div>

<!-- Center (horizontal) -->
Center an element horizontally.
<div class="ion-center"></div>

<!-- Circle -->
Make an element circular.
<div class="ion-circle"></div>

<!-- Horizontal Rule (hr) -->
<hr class="ion-hr">

<!-- Display only on small screen sizes -->
<div class="ion-visible-sm">...</div>
